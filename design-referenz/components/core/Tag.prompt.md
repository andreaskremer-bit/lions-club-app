Pill label for event types and categories. For an event's own registration status use `StatusBadge`.

```jsx
<Tag tone="blue">Club-Abend</Tag>
<Tag tone="gold" dot>Ehrenmitglied</Tag>
<Tag outline icon={<svg/* lucide map-pin */ />}>Bonn</Tag>
```

`tone` neutral|blue|gold|clay|sage, plus `outline`, `dot`, and a leading `icon`.
